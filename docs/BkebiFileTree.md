# Bkebi 文件介绍

| **文件名/文件夹名**      | **说明**       |
|:-----------------:|:------------:|
| injector\.exe     | Bkebi 本体注入器  |
| libstdimpact\.dll | Bkebi 本体依赖   |

| themes            | 主题           |
| cfg\.ini          | 注入器配置        |
| cfg\.json         | Bkebi 配置文件   |
| imgui\.ini        | Bkebi 内部窗口位置 |
